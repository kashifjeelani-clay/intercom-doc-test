# Weekday integration

Enhance lead generation efforts by providing and organizing additional contact information channels

The Weekday Integration enriches records by automating data retrieval with LinkedIn profile URLs.

It enhances lead generation efforts by providing and organizing additional contact information channels.

## **Enriching data with Weekday**

1.  While in a Clay table, click `Add enrichment` and search for `Weekday`.
2.  Under `Integrations`, select one of the Weekday options.
    -   If you have your own account, click `+ Add account` and go through authentication. Otherwise, use the Clay provided key.

### `Action` Find phone number

Use this action to find personal phone numbers for prospects using their LinkedIn profile URLs.

**Inputs**

-   **Professional profile URL:** The LinkedIn URL of the person you want to find a phone number for (e.g., `https://www.linkedin.com/in/john-doe-1234567890/`)

**Output**

-   **Phone Number:** The prospect's personal phone number (e.g., `+918080086433`)
-   **LinkedIn URL:** The professional profile URL used for the lookup
