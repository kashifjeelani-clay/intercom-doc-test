# TheirStack integration

Access current job openings and monitor hiring trends efficiently using company domains.

TheirStack Integration enables users to access current job openings and monitor hiring trends efficiently using company domains.

It supports lead generation by identifying actively hiring companies, facilitating competitive analysis.

## **Enriching data with TheirStack**

1.  While in a Clay table, click `Add enrichment` and search for `TheirStack`.
2.  Under `Integrations`, select one of the TheirStack options.

### **`Action` Find jobs**

The "Find jobs" action enables users to quickly access and review current job openings at a specific company using its domain, providing valuable insight into hiring activity.

**Inputs**

-   **Company domain:** Domain of the target company
-   **Company professional profile URL:** Professional profile URL of the target company
-   **Job title keywords to include (Optional):** Keywords to filter job titles
-   **Job title keywords to exclude (Optional):** Keywords to exclude from job titles
-   **Job description keywords to include (Optional):** Keywords to filter job descriptions
-   **Job description keywords to exclude (Optional):** Keywords to exclude from job descriptions
-   **Job location (Optional):** Location to filter job openings
-   **Technologies used (Optional):** Technologies to filter by
-   **Days since posted (Optional):** Number of days since the job was posted
-   **Job posted after date (Optional):** Filter for jobs posted after a specific date
-   **Job posted before date (Optional):** Filter for jobs posted before a specific date
-   **Employment statuses (Optional):** Employment status types to filter by
-   **Has recruiter (Optional):** Whether the job posting has a recruiter
-   **Include job descriptions (Optional):** Include full job descriptions in results
-   **Limit (Optional):** Maximum number of job openings to return
