# Stripe integration

Global payment processing platform enabling secure transactions and revenue growth.

## Stripe Overview

The Stripe integration allows you to retrieve detailed information about a specific subscription using its unique ID.

## Setting up Shopify and Clay integration

To start, connect your Stripe account by selecting **\+ Add account** within the Stripe enrichment panel. Ensure that your Stripe account credentials are set up in Clay to enable this action.

## How to retrieve Stripe subscription by ID within Clay

**Step 1: Select the Stripe action**

You can access the Stripe integration through the search panel.

**Step 2: Connect your Stripe account**

Connect your Stripe account by clicking **\+ Add account**

**Step 3: Enter Subscription ID**

Enter the ID of your Stripe subscription.

**Step 4: Configure run settings**

Auto-update: Stripe will automatically enrich any new rows that get added to the table. Learn more about auto-update in this [this brief guide](https://docs.clay.com/en/articles/9642165-auto-update-and-auto-dedupe-table).

Conditional runs: To run enrichment only under specific conditions, use formulas that trigger the column when the formula is true. See [this Clay University lesson](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101).
