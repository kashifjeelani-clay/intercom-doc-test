# Shopify integration

E-commerce platform for online stores and retail point-of-sale systems.

## Shopify Overview

The Shopify integration allows you to retrieve information about products on a Shopify-hosted website and check the hosting status of any store. This integration is useful for identifying if a website is hosted on Shopify and accessing product details directly from the store.

## Setting up Shopify and Clay integration

No setup is required. Clay credits will be used to access and enable the integration with Shopify.

## **Available Actions with the Shopify Integration**

### `Action` Find Products

Use this action to retrieve a list of products available on a Shopify-hosted website.

Setup Inputs

-   **Shopify Website URL**: Enter or select a column containing the URL of the Shopify store.
-   **Return Counts Only?** (Optional): Enable this setting if you only want to retrieve the high-level counts on the total number of products, collections, and vendors

### `Action`Check Store Status

This action verifies if a given site is hosted on Shopify, helping you confirm platform compatibility.

Setup Inputs

-   **Shopify Website URL**: Enter or select a column containing the website URL to check.
