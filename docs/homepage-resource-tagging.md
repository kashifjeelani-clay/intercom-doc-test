# Homepage Resource Tagging

Resource tags help you organize your Clay workspace by adding custom labels to folders, tables, workbooks, and functions. They make it easier to find content, share context with your team, and create flexible organizational systems.

You can add tags to:

-   Folders
-   Tables
-   Workbooks
-   Functions

## **When to use tags**

**Tags complement folders** by adding flexible cross-categorization to your workspace:

-   **Folders**: Create hierarchy and nest related workbooks (like file cabinets).
-   **Tags**: Label resources by status, team, or project across different folders (like sticky notes).

**Example**: A workbook might live in the "Q1 2026" folder, but have tags for "SDR Team," "Active," and "EMEA Region" to make it findable in multiple ways.

**Why use tags?**

-   Find resources faster by filtering across multiple dimensions.
-   Track resource status without creating complex folder structures.
-   Share context with teammates about what resources are for.
-   Organize by team, project, priority, or any system that works for you.

## **Managing tags**

**Admin-only feature:** Only workspace admins can create new tags. If you're not an admin, ask your workspace admin to create tags, then you can apply them to resources.

**To create a new tag** (admins only):

1.  Navigate to `Settings` → `Workspace settings`.
2.  Find the `Tags` section.
3.  Click `Create tag`.
4.  Enter a tag name (up to 30 characters).
5.  Optionally, select a color to help visually distinguish the tag.
6.  Click `Save`.

Once created, the new tag is immediately available to all workspace members.

### **Adding tags to resources**

All workspace members can assign existing tags to resources.

**Method 1: Right-click menu**

1.  Right-click on any folder, table, workbook, or function.
2.  Select `Tags` from the context menu.
3.  Choose from existing tags or search for specific tags.
4.  Tags are applied immediately.

**Method 2: Resource dropdown**

1.  Click the three-dot menu (⋮) on any resource card.
2.  Select `Tags`.
3.  Choose tags to apply.

**Method 3: Bulk selection**

1.  Hold `Shift` and click multiple resources to select them.
2.  Right-click on the selection.
3.  Select `Tags` and choose the tags you want to apply.
4.  All selected resources will be tagged at once.

**Tip:** You can add multiple tags to a single resource to categorize it in different ways.

### **Where tags appear**

Once you tag a resource:

-   Tags display on the resource card on your homepage.
-   Tags appear in the resource details sidebar.
-   You can filter your entire workspace by clicking tags in the Filter dropdown.
-   Tags show in search results to help identify resources.

### **Filtering by tags**

**From your homepage:**

1.  Look for the `Filter` option in the top navigation.
2.  Click `Tags` to see all available tags.
3.  Select one or more tags to filter your view.
4.  Only resources with the selected tags will be displayed.

**From your favorites:**

Tags can also be used to filter resources in your favorites tab.

All workspace members can see and filter by all tags in the workspace, regardless of who created them.

### **Editing tags (Admins only)**

Workspace admins can rename existing tags or change their colors:

1.  Go to `Settings` → `Workspace settings` →`Tags`.
2.  Click on the tag you want to edit.
3.  Update the tag name or color.
4.  Click `Save`.

Changes apply automatically to all resources using that tag.

### **Deleting tags (Admins only)**

When you delete a tag, it's automatically removed from all resources where it was applied.

To delete a tag:

1.  Go to `Settings` →`Workspace settings` →`Tags`.
2.  Find the tag you want to delete.
3.  Click the trash icon next to the tag.
4.  Confirm the deletion.

### FAQs

**Can I add multiple tags to one resource?**

Yes. You can add as many tags as you need to any resource.

**Are tags personal or workspace-wide?**

Tags are workspace-wide, meaning everyone in your workspace can see and use the same tags. Personal or private tags are not currently supported.

**Can I use tags across multiple workspaces?**

No, tags are scoped to individual workspaces. Each workspace has its own separate set of tags.

**Are there any default tags?**

No. Workspaces start with a blank tag library. Workspace admins can create custom tags to fit your team's needs.

**Can I use tags to trigger automations?**

Not at this time. Tag-based automation and workflows are not currently supported.

**What happens to resources when I delete a tag?**

When a workspace admin deletes a tag, it's automatically removed from all resources where it was applied. The resources themselves are not affected.

**Can I search for resources by tag?**

Yes. You can filter resources by tags from your homepage view or favorites tab. You can also search for resources using tag names.

**What's the difference between folders and tags?**

Folders create a hierarchical structure where workbooks live in one specific location. Tags are flexible labels that can be added to any resource regardless of which folder it's in, allowing you to find resources across multiple dimensions (by team AND status AND project, for example).
