# ElevenLabs integration

Use Text to Speech to convert text to a variety of voices.

ElevenLabs is a powerful AI tool for generating realistic text-to-speech audio.

With this integration, you can convert any text into natural-sounding speech using a variety of voices.

## Setting up the ElevenLabs integration

1.  While in a Clay table, click `Add enrichment` and search for ElevenLabs.
2.  Under `Integrations`, select one of the ElevenLabs options.
3.  In the modal, you will be asked to `Select ElevenLabs account`.
    1.  If you have your own account, click `+ Add account` and go through authentication. Otherwise, use the Clay provided key.

## Using the ElevenLabs integration

### `Action` Text to Speech

Generate high-quality, realistic speech from text.

**Inputs**

-   **Text (2500 characters maximum)**
-   **Voice:** Select a voice from your ElevenLabs account.
-   **Maximum credit spend (29 characters per credit) (Optional):** Set a maximum number of credits to spend per audio generation.

**Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
