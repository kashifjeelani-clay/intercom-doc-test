# Exellius integration

Find and verify contact information in Singapore and Japan.

Exellius is a data enrichment tool for finding and verifying contact information in Singapore and Japan.

With this integration, you can find business contacts, enrich contact details like phone numbers and emails, and gather company information for prospects in the APAC region.

## **Enriching data with Exellius**

1.  While in a Clay table, click `Add enrichment` and search for `Exellius`.
2.  Under `Integrations`, select one of the Exellius enrichment options.

### `Action` Find people

Search for people in Singapore and Japan.

**Inputs**

-   **Company professional profile URL (Optional)**
-   **Company domain (Optional)**
-   **Seniority (Optional)**
-   **Department (Optional)**
-   **Limit (Optional)**

### `Action` Enrich mobile phone

Search for phone numbers in Singapore and Japan.

**Inputs**

-   **Person professional profile URL (Optional)**
-   **Email address (Optional)**

### `Action` Enrich work email

Search for work emails in Singapore and Japan.

**Inputs**

-   **Person professional profile URL (Optional)**
-   **Full name (Optional)**
-   **Company domain (Optional)**

### `Action` Enrich work email

Search for work emails in Singapore and Japan.

**Inputs**

-   **Person professional profile URL (Optional)**
-   **Full name (Optional)**
-   **Company domain (Optional)**

### `Action` Enrich work email

Enrich company data for people in Singapore and Japan.

**Inputs**

-   **Company professional profile URL (Optional)**
-   **Company domain (Optional)**

### **Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
