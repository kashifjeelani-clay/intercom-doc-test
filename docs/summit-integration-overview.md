# Summit integration

IT service management enhances efficiency with automation and cost optimization.

## Summit Overview

Summit is an app in Clay that allows you to send data to an external application and receive a response, enabling seamless integration between Clay and other software systems.

## Setting up Summit and Clay integration

To set up the Summit integration, connect your Summit account by adding your API key. You can access adding accounts in the **Settings > Connections** or via the **Run App** integration panel.

For more information on accessing your Summit API key, please visit Summit’s [documentation](https://summit.readme.io/docs/triggers).

## **Available Actions with the Summit Integration**

### `Action` Run App

Use this action to push Clay data to an app in Summit and receive a response.

**Setup Inputs**

-   **App Name**: Enter or select a column containing the identifier of the app you want to run in Summit.
-   **Dynamic Fields** (Optional): Provide any additional fields required by the specific Summit app.
