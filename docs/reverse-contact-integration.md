# Reverse Contact integration

Find professional profiles with Reverse Contact.

Reverse Contact is a professional data enrichment tool for finding and validating contact information from email addresses.

With this integration, you can easily find professional profiles and additional details about your contacts directly within Clay.

## **Enriching data with Reverse Contact**

1.  While in a Clay table, click `Add enrichment` and search for `Reverse Contact`.
2.  Under `Integrations`, select one of the Reverse Contact options.
3.  In the modal, you will be asked to `Select Reverse Contact account`.
    -   If you have your own account, click `+ Add account` and go through authentication. Otherwise, use the Clay provided key.

### `Action` Find professional profile

Use this action to find a person's professional profile from their personal or work email.

**Inputs**

-   **Email**
-   **Full name (Optional)**
-   **Company name (Optional)**
-   **Company domain (Optional)**
-   **Include full details (Optional):** If switched on, Reverse Contact will return the full details of the enriched person, including their past experiences and education.

### **Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
