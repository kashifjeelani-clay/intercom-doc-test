# Linear integration overview

Streamlined project management software for teams focused on product development workflows.

## Linear Overview

Linear is a tool for planning and managing software development projects.

You can streamline issue management in Clay by quickly finding or creating Linear issues without leaving your workflow.

## Setting up Linear and Clay integration

To set up the Linear integration, connect your Linear account by signing in via OAuth. You can access this in the **Settings > Connections** or via any integration panel.

## **Available Actions with the Linear Integration**

### `Action` Create Issue

Use this action to create an issue in Linear directly from Clay.

**Setup Inputs**

-   **Title**: Enter or select a column containing the title of the issue you’d like to create in Linear.
-   **Description** (Optional): Provide a detailed description of the issue. Markdown formatting is supported.
-   **Team**: Specify the team to which the issue belongs.

### `Action` Find Issue

Use this action to find an issue in Linear based on a search query directly from Clay.

**Setup Inputs**

-   **Title** (Optional): Enter or select a column containing the title of the issue you’d like to search for in Linear.
-   **Description** (Optional): Provide a description to narrow down the search. Markdown formatting is supported.
-   **Team** (Optional): Specify the team associated with the issue to refine the search.
