# Oakie.ai integration

Extract key insights from public company filings

[Oakie.ai](http://Oakie.ai) is an AI-powered platform that analyzes SEC filings and financial documents to provide competitive intelligence and market insights.

With this integration, you can extract key insights from public company filings directly in your Clay table.

## Enriching data with [Oakie.ai](http://Oakie.ai)

1.  While in a Clay table, click `Add enrichment` and search for `Oakie.ai`.
2.  Under `Integrations`, select one of the [Oakie.ai](http://Oakie.ai) options.

### `Action` AI Insights from Public Company Filings

Use AI to analyze and ask questions about companies' public financial filings (10-K, 10-Q).

**Inputs**

-   **Company name / description / stock ticker**
-   **Question**
-   **Start date for filings**
-   **End date for filings**
-   **Filing type to search**
