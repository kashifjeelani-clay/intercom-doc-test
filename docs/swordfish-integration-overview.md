# Swordfish integration

Contact data search engine providing accurate cell phone and email information for sales and recruitment.

## Swordfish Overview

Swordfish is an app in Clay that enriches a person's profile with additional data using advanced AI technology.

## Setting up Swordfish and Clay integration

To set up the Swordfish integration, connect your Swordfish account by adding your API key. You can access the Account Add in the **Settings > Connections** or via any integration panel.

## **Available Actions with the Swordfish Integration**

### `Action` Enrich Person

Use this action to retrieve a list of products available on a Swordfish-hosted website.

**Setup Inputs**

-   **Must Have**: Specify the data that must be found for the search to cost credits in Swordfish (defaults to “Any Email”).
-   **Full Name**, **Email**, **Phone**, **Social Profile**, **City**, **State**, **Country**, **Zip Code** (Optional): Provide any known details to enhance the enrichment results.
