# RB2B integration

E-commerce platform to re-engage visitors, enhance retention, and grow revenue.

RB2B is a data provider that allows you to create personalized AI videos with custom scripts for engaging, user-generated marketing and sales content.

With this integration, you can find personal email addresses associated with professional social profiles.

## Setting up the RB2B integration

1.  While in a Clay table, click `Add enrichment` and search for RB2B.
2.  Under `Integrations`, select the RB2B option.
3.  In the modal, you will be asked to `Select RB2B account`.
    -   Use the Clay provided key (4 credits per enriched cell).

## Using the RB2B integration

### `Action` Find Personal Email from LinkedIn URL

Use this action to find a personal email associated with a professional profile.

**Inputs**

-   **Social URL**: Select a column containing the professional social URL of the person you want to find. This can be a standard or Sales Navigator URL.

**Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
