# HeyGen integration

Generate videos with AI avatars using your scripts.

HeyGen is a powerful AI video generation tool for creating professional avatar videos.

With this integration, you can easily generate high-quality videos featuring AI avatars speaking your custom scripts.

## Setting up the HeyGen integration

1.  While in a Clay table, click `Add enrichment` and search for HeyGen.
2.  Under `Integrations`, select one of the HeyGen options.
3.  In the modal, you will be asked to `Select HeyGen account`.
    1.  If you have your own account, click `+ Add account` and go through authentication. Otherwise, use the Clay provided key.

_Note: If you want to use a custom avatar, authenticate with your own HeyGen account._

## Using the HeyGen integration

### `Action` Generate avatar video

Use this action to create high quality AI avatar videos.

**Inputs**

-   **Avatar image**
-   **Avatar voice**
-   **Script for Avatar video**
-   **Background type**
-   **Aspect ratio (Optional)**
-   **Avatar style (Optional)**
-   **Avatar scale (Optional)**
-   **Avatar position (Optional)**
-   **Voice speed (Optional)**

**Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
