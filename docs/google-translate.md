# Google Translate integration

Translate text, documents, and websites across over 100 languages

Google Translate is a powerful tool for translating text between languages.

With this integration, you can translate text in your Clay tables from one language to another.

## **Enriching data with Google Translate**

1.  While in a Clay table, click `Add enrichment` and search for `Google Translate`.
2.  Under `Integrations`, select one of the translation options.
3.  In the modal, you will be asked to `Select Google Translate account`.
    -   If you have your own account, click `+ Add account` and go through authentication. Otherwise, use the Clay provided key.

### `Action` Google Translate

Translate text with Google Translate.

**Inputs**

-   **Enter Text**
-   **Target Language**
-   **Source Language (Optional):** If left empty, the language will be auto-detected based on the source text.

### **Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
