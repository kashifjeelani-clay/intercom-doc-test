# Google PageSpeed integration overview

Web performance analysis tool.

## Google PageSpeed Overview

Google PageSpeed provides performance insights for websites, analyzing factors like loading speed and optimization opportunities for both mobile and desktop views. In Clay, you can use Google PageSpeed to monitor and improve website performance, helping to identify areas for optimization based on Google’s recommendations.

## Setting up Google PageSpeed and Clay integration

To set up the Google PageSpeed integration, connect your Google PageSpeed account by signing in via OAuth. You can access this in the **Settings > Connections** or via any integration panel.

## **Available Actions with the Google PageSpeed Integration**

### `Action` **Find Data with Google PageSpeed**

Use this action to retrieve a list of products available on a Google PageSpeed-hosted website.

**Setup Inputs**

-   **Google PageSpeed Website URL**: Enter or select a column containing the URL of the Google PageSpeed store.
-   **Return Counts Only?** (Optional): Enable this setting if you only want to retrieve the high-level counts on the total number of products, collections, and vendors
