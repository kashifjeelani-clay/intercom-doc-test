# Rox integration

Update contacts in Rox directly from Clay

Rox is a contact management platform that helps teams organize and update contact information efficiently.

With this integration, you can update contacts in Rox directly from Clay.

## **Enriching data with Rox**

1.  While in a Clay table, click `Add enrichment` and search for `Rox`.
2.  Under `Integrations`, select one of the Rox options.
3.  In the modal, you will be asked to `Select Rox account`.
    -   If you haven't already connected your Rox account, click `+ Add account` and go through authentication.

### `Action` Update Rox contact

Use this action to update existing contact records in Rox.

**Inputs**

-   **Rox org ID**
-   **Rox user ID**
-   **Rox person ID**
-   **Company domain**
-   **Professional profile URL**
-   **Email (Optional)**
-   **Full name (Optional)**
-   **Job title (Optional)**
-   **Current company (Optional)**
-   **Country (Optional)**
-   **Location (Optional)**
-   **Profile picture URL (Optional)**
-   **Seniority (Optional)**
-   **Snippet (Optional)**
-   **Snippet sources (Optional)**
-   **Persona (Optional)**
-   **Phone number (Optional)**
-   **Employment history (Optional)**

### **Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
