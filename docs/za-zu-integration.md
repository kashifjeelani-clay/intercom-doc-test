# Za-zu integration

Lookup and add leads to your campaign.

Za-zu is a lead management tool for marketing teams.

With this integration, you can lookup and add leads to your Za-zu campaigns directly from Clay.

## Setting up the Za-zu integration

1.  While in a Clay table, click `Add enrichment` and search for Za-zu.
2.  Under `Integrations`, select one of the Za-zu options.
3.  In the modal, you will be asked to `Select Za-zu account`.
    1.  If you haven't already connected your Za-zu account, click `+ Add account` and go through authentication.

## Using the Za-zu integration

### `Action` Lookup Lead

Use this action to lookup a lead.

**Inputs**

-   **Email**
-   **Campaign ID (Optional)**

**Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))

### `Action` Add Lead to Campaign

Use this action to add a lead.

**Inputs**

-   **Campaign ID**
-   **Email**
-   **First name (Optional)**
-   **Last name (Optional)**
-   **Social profile URL (Optional)**
-   **Job title (Optional)**
-   **Company name (Optional)**
-   **Phone number (Optional)**
-   **New custom fields (Optional)**

**Run settings**

-   **Auto-update**
-   **Only run if:** The enrichment will only run if conditions are met. ([Learn more about conditional formulas here!](https://www.clay.com/university/lesson/ai-formulas-conditional-runs-clay-101))
