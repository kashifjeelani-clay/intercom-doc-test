# Signalhire integration

Finds contact details for recruiting and sales.

## **SignalHire Overview**

The SignalHire integration in Clay allows users to search for detailed candidate information, including contact details, work history, and social profiles, using LinkedIn URLs, email addresses, or phone numbers.

## Setting up SignalHire and Clay integration

To set up the SignalHire integration, connect your SignalHire account by integrating your API key into Clay. You can access this in the **Settings > Connections** or via any integration panel.

## **Available Actions with the SignalHire Integration**

### **`Action` Search for a Candidate**

Use this action to find detailed information about a candidate by providing LinkedIn URLs, email addresses, or phone numbers.

**Setup Inputs**

-   **Search Items**: Enter one or more identifiers (LinkedIn URL, email address, or phone number in E164 format, e.g., +15552223333). Multiple values can be entered, separated by a comma, or as an array of values.
